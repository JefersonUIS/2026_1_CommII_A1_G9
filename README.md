# CommII_A1_G9

Bienvenido al informe de la Práctica 1 de Comunicaciones II, donde se abordó la implementación de bloques de Python personalizados en GNU Radio para Radio Definida por Software (SDR), integrando algoritmos de acumulación, diferenciación y análisis estadístico en tiempo real. Este trabajo nos permitió validar modelos teóricos mediante la caracterización de señales bajo diversas tasas de muestreo y entornos ruidosos, demostrando la versatilidad de los bloques programables para el procesamiento digital de señales y la medición de parámetros como media, RMS y potencia.
